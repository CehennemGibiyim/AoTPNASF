# 📦 Kurulum

## Repository Klonlama

```bash
git clone https://github.com/CehennemGibiyim/AoTPNASF.git
cd AoTPNASF
```

## Local Sunucu Başlatma

### Python

```bash
python -m http.server 8000
```

### Node.js

```bash
npx http-server -p 8000
```

## Tarayıcıdan Aç

```txt
http://localhost:8000
```
